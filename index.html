CSS Common Properties Cheat Sheet

Text Properties

Tag Description
color: Set the text color.
font-family: Define the font used for the text.
font-size: Specift the size of the font.
font-weight: Control the thickness of the font.
text-align: Align the text within its container.

Box Model Properties

Tag Description
width: Set the width of an element.
height: Define the height of an element.
margin: Create space around an element.
padding: Add space inside an element.
border: Customize the border around an element

Layout Properties

Tag Description
display: Define how an element should be displayed.
position: Specify the positioning method of an element.
float: Control the float behavior of an element.
z-index: Manage the stacking order of elements.
overflow: Handle content overflow within an element.
